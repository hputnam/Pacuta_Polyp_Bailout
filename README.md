## Project Overview 
This repository contains data, scripts and analysis from a study investigating polyp bail-out in the scleractinian
coral Pocillopora acuta as a result of salinity induced stress. The research was conducted at the University of Rhode Island (URI) on Kingston Campus in Rhode Island during the year 2025 and 2026.
